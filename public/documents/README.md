# Documents Directory

Place PDF files here with the following naming convention:

- `payer-guidelines-{payerName}.pdf` - Payer-specific guidelines
- `clinical-criteria.pdf` - Clinical criteria documentation
- `cpt-codes.pdf` - CPT codes reference
- `nar-grid.pdf` - No Authorization Required grid

Example files:
- payer-guidelines-aetna.pdf
- payer-guidelines-bcbs.pdf
- payer-guidelines-cigna.pdf
- clinical-criteria.pdf
- cpt-codes.pdf
- nar-grid.pdf

These will be accessible at: `/documents/{filename}`
